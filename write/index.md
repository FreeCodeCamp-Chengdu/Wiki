## 多媒体文件上传

图片、视频、PDF、PPT 等**二进制文件**上传必须在[本地编辑][1]！

## Front-matter 规范

（点右下角按钮复制模板）

```yaml
title: # 文章主题（翻译成英文后设为本文件的文件名）
date: # 开篇日期
updated: # 修订日期
categories:
  # 分类名应为一个首字母大写的英文单词
  # 每级分类对应一级目录，写在本文件的文件名前面
tags:
  # 与内容相关的更多标签
toc: true
authors:
  # 原作者们的 GitHub 账号
```

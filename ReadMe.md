## 本机编辑

为了加快图片、视频等**大型二进制文件**的传输速度，本仓库启用了 [Git LFS][2]，请确保在 `git clone` 前装好 LFS 扩展：

```shell
# Windows - Chocolatey
choco install git-lfs

# Mac OS X - HomeBrew
brew install git-lfs
```
